# Python Quant Foundations — Problem Sets

Ce dossier contient des mini-problèmes pour t'entraîner comme dans CS50P, mais orientés finance quantitative.

Objectif : renforcer les bases Python avant d'aller plus loin vers NumPy, simulations, returns, risk metrics et Monte Carlo.

## Méthode de travail

Pour chaque fichier :

1. Lis l'énoncé dans la docstring.
2. N'écris pas directement une solution compliquée.
3. Commence avec une version simple qui marche.
4. Lance le fichier dans le terminal.
5. Teste plusieurs inputs.
6. Crée ensuite une version `_practice.py` sans regarder.
7. Ajoute une petite amélioration.
8. Commit et push.

Exemple :

```bash
cd ~/python-quant-foundations
source .venv/bin/activate
python 00_problem_sets/01_conditionals.py
```

## Règles

- Pas de pandas pour ces problèmes.
- Pas de NumPy sauf quand le problème le demande explicitement.
- Pas de ChatGPT pour copier une solution complète.
- Tu peux demander des indices, des corrections ou une explication d'erreur.
- Chaque problème doit être compris ligne par ligne.
